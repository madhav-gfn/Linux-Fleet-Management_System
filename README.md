# Linux Fleet Management Lab

## Overview
A local Linux fleet management and monitoring lab built on VirtualBox,
simulating real-world infrastructure practices.

## Architecture
- 1 Manager node
- 2 Worker nodes
- SSH-based control plane
- Prometheus + Grafana monitoring stack

## Stack
- Ubuntu Server 22.04 LTS
- OpenSSH
- Prometheus
- Node Exporter
- Grafana

## What This Project Demonstrates
- Secure node onboarding using SSH keys
- Agent-based observability
- Pull-based metrics collection
- Realistic fleet-style architecture
